# "Акинатор"
Разработать консольное приложение на языке C, реализующее алгоритм «Акинатор» — угадывание объекта по ответам «да»/«нет» на основе бинарного дерева вопросов..

Все команды выполняются в командной строке разработчика (по крайней мере для VS)
## Как собрать
Команда nmake all

## Как запустить
bin\akinator.exe

## Как запустить тесты
Команда nmake test

## Как очистить 
Команда nmake clean

## Формат файла дерева
("корневой вопрос"
    (нет -> "объект А")
    (да -> ("второй вопрос"
                (нет -> "объект Б")
                (да -> "объект В")))

## Структура проекта
src/ – исходный код
include/ – публичные заголовки
data/ – файлы с деревьями (примеры)
tests/ – модульные тесты
docs/ – документация (Doxygen)
Makefile – сценарий сборки"
